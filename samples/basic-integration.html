Basic Chatbot Integration

This page demonstrates the simplest way to integrate the AI chatbot widget.

Integration Code

<script src="../dist/chatbot-widget.min.js"></script>
<script>
  ChatbotWidget.init({
    apiUrl: 'http://localhost:5000',
    botName: 'AI Assistant'
  });
</script>

Click the chat icon in the bottom right corner to start a conversation.
